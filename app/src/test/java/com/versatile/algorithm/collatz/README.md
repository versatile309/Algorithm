## 01. 이해
    정수를 입력 받아 콜라츠 추측이 성립될 때까지 몇번의 연산이 필요한지를 반환
    만약 500번안에 성립되지 않으면 -1을 반환
       
## 02. 계획
    1이 될때까지 짝수면 나누기 2, 홀수면 *3+1을 반복해서 연산 횟수를 반환한다.
    도중에 연산 횟수가 500이 되면 -1을 반환한다.
