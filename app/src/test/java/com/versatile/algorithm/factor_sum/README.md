## 01. 이해
    정수의 약수들의 합을 리턴
       
## 02. 계획
    약수를 구하고, 합치면 됨.
    a이를 b로 나눴을 때 나머지가 0이면 b는 a의 약수인 거니까
    1부터 입력받은 정수까지의 숫자들 중에 입력받은 정수를 나눴을 때 나머지가 0인
    정수만 더해서 반환
