## 04. 회고
    단순히 문자열로 만들고 정렬하니까 3과 30이 있을 때 3이 먼저 와야하는데
    30이 먼저 와버리는 문제가 발생해서 결국 두 문자열을 앞뒤로 각각 붙여서 비교하는
    방법으로 정렬을 진행했다.
    만약 모든 원소가 0일 경유 "000.."이런 문자열이 나올 수 있어서 이 경우에는 "0"을
    반환하도록 예외 처리  했다.
