## 01. 이해
    현재 작업별 진행도 배열과 작업별 진행속도 배열을 입력 받아 각 배포 때 
    몇개씩 배포가 진행되는지 반환. 단 앞의 작업이 완료되 지 않으면 뒤의 작업은
    완료되더라도 배포될 수 없음
       
## 02. 계획
    작업 배열을 계속 돌면서 진행 속도를 더하다가 맨 앞의 작업이 100이 되었을 때
    앞쪽 부터 100이 넘은 작업을 빼내고 그 갯수를 결과 배열에 넣기.
