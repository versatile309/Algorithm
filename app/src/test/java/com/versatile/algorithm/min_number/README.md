## 01. 이해
    받은 정수 배열에서 가장 작은 수를 제거하고 반환한다.
    다만 리턴하는 배열의 크기가 0일 경우에는 -1만 들어있는 배열을 받환 한다.
       
## 02. 계획
    배열의 크기가 1일 경우에 가장 작은 수를 제거하면 무조건 빈 배열이 되기 때문에
    이 경우엔 무조건 -1만 들어있는 배열을 반환한다.
    이 외의 경우엔 filter를 통해 가장 작은 수를 제거하고 반환 한다.
