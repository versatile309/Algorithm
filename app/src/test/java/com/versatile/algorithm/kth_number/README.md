## 01. 이해
    정수 배열과 길이 3의 커맨드로 이루어진 2차원 커맨드 배열을 받는다. 
    정부 배열을 각 커맨드별 첫번째 인덱스의 수부터 두번째 인덱스의 수까지로 자르고, 정렬하고,
    이중에 세번째 인덱스 번째에 있는 숫자를 모아서 배열로 반환
       
## 02. 계획
    2차원 커맨드 배열에서 차례대로 커맨드를 읽는다.
    정수배열을 슬라이스, 정렬, 커맨드에 맞춰 숫자를 찾는다.
    정답 배열에 숫자를 넣는다.
